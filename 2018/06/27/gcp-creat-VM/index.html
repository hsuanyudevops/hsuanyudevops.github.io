建立GCP-VM(GUI)

Jun 27, 2018

1. 進入GCP首頁:https://cloud.google.com/

2. 登入後點擊畫面右上角控制台進入

3. 建立新專案

打上專案名稱後點擊建立

4. 建立VM

(1)導覽選單→(2)Compute Engine→(3)VM執行個體

點擊建立VM

輸入VM"名稱"後，按照需求可選擇建置在哪個"區域"下，"機器類型"可按照他的預設規格做選擇，也可以點擊"自訂"自行調正CPU核心數及記憶體大小，"開機磁碟"點擊變更可選擇Linux、Windows系統，以及開機磁碟的大小、磁碟類型(HDD，SSD)，"防火牆"的部分按照需求勾選允許HTTP、HTTPS流量。 在"磁碟"下方點選”新增項目”新增資料碟

在"網路"欄位下方展開網路介面的選項，若內外有需要使用靜態IP，在主要內部IP選單選擇”預約靜態IP”，在外部IP選單選擇”建立IP位址”，若沒有此需求，使用臨時即可。

完成以上設定後至頁面最下方點選”建立”即可完成VM建置

關於GCP計費說明可參考官網:https://cloud.google.com/compute/pricing#internet_egress
費用估算可參考:https://cloud.google.com/products/calculator/
