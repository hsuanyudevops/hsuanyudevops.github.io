Jul 03, 2018

進入GCP首頁:https://cloud.google.com/

登入後點擊畫面右上角控制台進入:

在控制台右上角工具欄中點擊Google Cloud Shell圖示:

Note: gcloud是Google Cloud Platform命令工具，完整文件可參閱官方說明:https://cloud.google.com/sdk/gcloud/

建立projects:

gcloud projects create <專案ID> –name=<專案名稱>

建立VM:

指令詳解:

instances create <VM名稱> –zone=<建置區域>

-1為CPU核心數
-2048為RAM大小(MB)

可使用下方指令查詢所要安裝的系統，再將所選擇系統的NAME及PROJECT填入
gcloud compute images list

Command output

–boot-disk-size=10GB 開機磁碟大小為(10GB)
–boot-disk-device-name=test01 開機磁碟名稱為(test01)

可使用下方指令查詢要使用的開機磁碟類型:
gcloud compute disk-types list

Command output

●PD，又稱為標準磁碟(Standard Disk)，為SATA格式之硬碟，提供大部分情境之作業系統或儲存使用。
●SSD，固態硬碟，具備較高的IOPS，當然，價格也較昂貴。
●Local SSD，本機SSD硬碟，具備GCE上最快的IOPS選擇。每次選購，必須完整租用整個Local SSD 375G的大小空間。在Local SSD的選用中，具備SCSI與NVMe兩種介面可以選擇。

建立資料碟:
10GB(容量)
pd-ssd(標準SSD)
test=disk(資料碟名稱)

先佔模式:實惠的短期運算執行個體，適合批次作業與容錯工作負載。
可參考官方文件:https://cloud.google.com/preemptible-vms/
